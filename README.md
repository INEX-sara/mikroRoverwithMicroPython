# ซอร์สโค้ดประกอบคู่มือ mikroRover Activity Guide

แหล่งเก็บรวบรวมซอร์สโค้ดตัวอย่าง (Listing) สำหรับหุ่นยนต์ **mikroRover** ที่ใช้งานร่วมกับบอร์ด **KidMotor V4i**

## โครงสร้างโปรเจกต์ (แยกตามบทเรียน)

* Chapter 03: Hello OLED** - การเริ่มต้นใช้งานจอ OLED 0.96" และการแสดงผลข้อความ 
* Chapter 04: Robot Movement** - พื้นฐานการควบคุมมอเตอร์และการเขียนฟังก์ชันเคลื่อนที่ 
* Chapter 05: Ultrasonic Sensor** - การใช้งานเซนเซอร์วัดระยะทาง ZX-SONAR1M เพื่อหลบหลีกสิ่งกีดขวาง 
* Chapter 06: Line Tracking** - การใช้งานเซนเซอร์อินฟราเรด ZX-03 เพื่อทำให้หุ่นยนต์วิ่งตามเส้นสีดำ 
* Chapter 07: Gripper-X Mission** - การควบคุมเซอร์โวมอเตอร์เพื่อใช้งานมือจับ และภารกิจยกของ (Move the can)
